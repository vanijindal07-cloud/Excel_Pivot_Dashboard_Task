## Task Details

1. **Data Cleaning:**  
   - Removed blanks and inconsistencies.  
   - Converted data into Excel Table format for dynamic pivot updates.  

2. **Pivot Tables Created:**  
   - Total Sales by Category  
   - Sales by State and Category with Conditional Formatting  
   - Average Profit Margin by Category and State  

3. **Pivot Chart Created:**  
   - Sales distribution by category for visual comparison.  

4. **Slicers Added:**  
   - Category, State, and Order Status for interactive filtering.  

5. **Insights Generated:**  
   - Key findings from all pivot tables and charts summarized in a separate document.

## Purpose

- To demonstrate **data analysis skills** using Excel pivot tables and charts.  
- To create a **professional, interactive dashboard** for business insights.  
- To generate clear, actionable **analytical storytelling** based on sales data.
